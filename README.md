# VALYOR — Premium Streetwear E-Commerce Frontend

VALYOR is a premium luxury streetwear e-commerce frontend inspired by modern fashion platforms like H&M and Zara. The project was designed and developed using HTML5, CSS3, and JavaScript with a strong focus on responsive design, cinematic visuals, interactive UI/UX, and modern frontend architecture. The website delivers a realistic high-end fashion shopping experience with immersive layouts, smooth transitions, and luxury-inspired aesthetics.

The project features a full-screen autoplay hero video section, responsive mega-menu navigation, promotional editorial grids, horizontally scrollable product carousels, hover animations, and a modern sticky navigation bar. The design follows a minimalist oversized streetwear theme with clean typography, strong visual hierarchy, and polished interaction effects to create a premium digital shopping experience.

The frontend structure is built using semantic HTML5 elements such as header, nav, section, video, and footer for clean organization, accessibility, and maintainability. CSS3 techniques including Flexbox, CSS Grid, media queries, z-index layering, transitions, backdrop blur effects, responsive scaling, and hover animations were used to create modern responsive layouts across desktop, tablet, and mobile devices.

Vanilla JavaScript was implemented to add interactivity and improve user experience. Features include smooth horizontal product carousel scrolling, newsletter form validation, dynamic video sound toggle functionality, event listeners, DOM manipulation, and responsive interaction handling without using external frameworks or libraries.

Key features of the project include:

* Fully responsive design
* Cinematic autoplay hero video
* Interactive mega dropdown navbar
* Product carousel with smooth scrolling
* Modern luxury fashion UI/UX
* Responsive promotional fashion grid
* Sticky navigation bar
* Smooth hover animations and transitions
* Newsletter subscription section
* Mobile-friendly responsive optimization

This project helped improve skills in frontend architecture, responsive web design, advanced CSS styling, interactive UI development, DOM manipulation, visual hierarchy, and modern e-commerce interface design. It was built as a portfolio-ready real-world frontend project to demonstrate practical development skills and modern UI/UX implementation techniques.

Tech Stack: HTML5, CSS3, JavaScript, Flexbox, CSS Grid, Responsive Web Design, Font Awesome Icons.
